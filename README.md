# Momentum

A web-based fitness and wellness tracking system developed as part of the HDip in Software Development at the National College of Ireland.

## Tech Stack

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MySQL

## Features

- User registration and login
- Workout plan creation and management
- Exercise logging (sets, reps, and weight)
- Personal record tracking
- Wellness tracking (mood, sleep, and energy)
- Progress dashboard and charts
